# 自动化测试Demo 项目

一个简单的自动化测试示例，使用：
- Playwright：Web UI自动化测试（百度搜索页面）
- Requests：REST API测试（JSONPlaceholder公开接口）
- Pytest：测试组织、断言、报告生成

## 运行方式
1. 安装依赖：`pip install -r requirements.txt`
2. 初始化Playwright浏览器：`playwright install`
3. 运行测试：`pytest`

## 生成报告
- HTML报告：`report.html`
- 截图保存在当前目录

目标：练习Web UI + API自动化测试基础，适用于SDET/自动化测试实习。
